# Contactify Frontend

This is a production-grade Next.js 15 project setup.

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- Zustand
- TanStack Query v5
- React Hook Form + Zod
- Shadcn UI

## Getting Started

First, install dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
